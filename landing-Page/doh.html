What is DNS?

The great translator.

The DNS server is a service whose main function is to translate a domain name into an IP address. Basically, the DNS server acts like a directory that a computer consults when accessing another computer over a network. In other words, the DNS server is this service which makes it possible to associate a website (or a connected computer or a server) with an IP address, as a telephone directory makes it possible to associate a telephone number with a subscriber name.

The exchange.

A domain name typed by a user in a browser (such as www.google.com) is translated by a DNS server into an IP address (216.58.201.238). This allows the device to find the resource you are looking for on the internetâ€”in this case, the google home page.

What is DoH/DoT?

DoH/DoT vs Unsecured.

DNS requests are currently carried out in the clear, which poses major confidentiality and security problems. Governments use lying DNS to perform censorship and hackers use infected routers to redirect traffic to where they want and ISPs use it to limit access based on subscription.

DNS over TLS uses the TCP protocol to establish the connection and it uses TCP port 853, a dedicated port. DNS packets are not modified, they are encrypted using the TLS protocol before being transmitted.
DNS over HTTPS uses the HTTPS protocol to establish the connection and it uses TCP port 443, a default port. DNS data is encapsulated in HTTP packets, which are encapsulated using the TLS protocol before being transmitted.
