What is DNS?

The great translator.

The DNS server is a service whose main function is to translate a domain name into an IP address. Basically, the DNS server acts like a directory that a computer consults when accessing another computer over a network. In other words, the DNS server is this service which makes it possible to associate a website (or a connected computer or a server) with an IP address, as a telephone directory makes it possible to associate a telephone number with a subscriber name.
